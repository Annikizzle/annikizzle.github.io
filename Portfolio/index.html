Game of Thrones RPG Game

The goal of this project was to create a functional turn-based Role-Playing Game in JavaScript. The user chooses a character and an opponent and they take turns fighting until a winner is chosen. The user/player then continues to fight until three enemies have been deafeated--once three enemies have been brought down, the player wins!

Queen Trivia Game

For this project, I wanted to create a fun trivia game that would also provide a learning experience. Many people know Queen--but not many people know about the member's lives, or the band's humble beginnings. Each question has a 15 second time limit, and when time is up, it proceeds to the next question.
